# DataPurge

DataPurge is a professional-grade data management dashboard designed to detect, analyze, and resolve redundancy in large-scale datasets. By automating the identification and purging of duplicate or "probable" redundant nodes, DataPurge optimizes cloud storage efficiency and ensures enterprise-grade data integrity.

## Features

* **Intelligent Ingestion Pipeline:** Streamlined CSV ingestion engine for real-time data monitoring.
* **Collision Detection:** Multi-stage processing to identify exact duplicates and probable redundant entries.
* **Resolution Center:** Dedicated dashboard for administrative review and bulk data resolution.
* **Audit Trail:** Real-time logging to maintain data integrity and transparency during cleanup operations.
* **Modern UI:** Responsive, dark-mode dashboard built for high-density information display.

## How to Run

1. **Clone the repository:**
```bash
   git clone [https://github.com/Bolzeii/CodeAlpha_DataPurge.git](https://github.com/Bolzeii/CodeAlpha_DataPurge.git)

```


2. **Open the application:** Navigate to the project folder and double-click `index.html` to open it in your preferred web browser.
3. **Ingest Data:** Navigate to the "Data Ingestion" tab and drag-and-drop your `.csv` file to begin processing.

## Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+).
* **Storage:** Browser-native IndexedDB for high-speed, local data persistence.
* **Data Handling:** Custom-built CSV parsing and pipeline processing logic.
